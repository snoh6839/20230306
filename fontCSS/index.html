글꼴 관련 스타일

font-family

기본 제공 폰트

폰트 패밀리 나눔 명조

넣은 폰트가 없을 경우 기본 제공 폰트가 표시됨
콤마로 구분해서 폰트를 추가로 넣어줄 경우 앞의 폰트가 없을 때
뒤의 폰트가 적용된다.

font-size

기본 폰트 사이즈 = 16px

키워드 타입 large

키워드 타입 small

폰트 사이즈 직접 지정 20px

폰트 사이즈 직접 지정 2em
em = 부모 폰트 사이즈의 배율 (부모 div = 16px, 2em = 32px)

폰트 사이즈 직접 지정 2rem
rem = 최상위 요소의 배율 (최상위 html = 10px, 2em = 20px)

font-weight

키워드 타입 bold
